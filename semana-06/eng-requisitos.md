## Requisitos da Aplicação

### Requisitos Funcionais

- **RF1** - O sistema deve enviar alertas em tempo real sobre chuvas fortes e alagamentos.
- **RF2** - O sistema deve exibir no mapa as áreas afetadas por alagamentos.
- **RF3** - O sistema deve permitir que usuários reportem alagamentos com foto e localização.
- **RF4** - O sistema deve sugerir rotas alternativas seguras ao usuário.
- **RF5** - O sistema deve enviar notificações com base na localização do usuário.
- **RF6** - O sistema deve disponibilizar um histórico de áreas afetadas.

### Requisitos Não Funcionais

- **RNF1** - O sistema deve enviar notificações em tempo real com baixa latência.
- **RNF2** - O sistema deve apresentar interface intuitiva e de fácil uso.
- **RNF3** - O sistema deve garantir a disponibilidade do serviço mesmo em situações críticas (chuvas intensas).
- **RNF4** - O sistema deve garantir a precisão das informações de localização.
- **RNF5** - O sistema deve suportar múltiplos usuários simultaneamente.
